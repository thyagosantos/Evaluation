# Solução CDB

Esta é uma solução que calcula o rendimento de um investimento em CDB.

## Configuração

- Certifique-se de ter o Visual Studio  e o Angular CLI e Node instalados
- Abra a solução no Visual Studio e execute a Web API.
- No diretório da aplicação Angular, execute `ng serve` para iniciar o servidor de desenvolvimento.

## Uso

- Acesse a aplicação Angular no navegador em: http://localhost:4200/investimento .
- Insira o valor inicial e o prazo do investimento.
- Clique no botão "Calcular Investimento" para ver os resultados.

## Testes
- Para executar testes unitários na Web API, abra o Visual Studio e utilize Tes explorer no projeto API.CDB.Tests.
- Para executar testes unitários no Angular, execute `ng test`.
